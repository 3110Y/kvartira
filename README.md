# ОСС — материалы общего собрания собственников

Этот репозиторий содержит документы и исходные данные, подготовленные для проведения **общего собрания собственников помещений** многоквартирного дома (далее — ОСС).

## Структура

```
.
├── README.md                          ← этот файл
├── poyasnitelnaya_zapiska.md          ← пояснительная записка для жильцов (готова)
└── data/                              ← исходные данные собрания
    ├── meeting.md                     ← параметры собрания (форма, инициатор, даты)
    ├── agenda.md                      ← повестка дня и принятые по каждому вопросу позиции
    ├── candidates.md                  ← кандидаты в совет дома
    └── references.md                  ← ссылки на внешние документы (Google Docs)
```

## Статус документов

| Документ | Статус |
| --- | --- |
| Пояснительная записка для жильцов | ✅ Готова — `poyasnitelnaya_zapiska.md` и [Google Doc](https://docs.google.com/document/d/1hd2a3kF04jfNj9GID5Qe09DCZqfNnHG4BMGJoiRrX4c/edit) |
| Бюллетень для голосования (опросник) | ⏳ В работе |
| Уведомление о проведении ОСС | ⏳ Не начато |
| Реестр собственников | ⏳ Не начато |
| Протокол ОСС (шаблон) | ⏳ Не начато |

## Что ещё нужно уточнить

Перед рассылкой бюллетеней по дому необходимо определить:

- адрес многоквартирного дома (для шапки документов);
- даты начала и окончания приёма заполненных бюллетеней;
- место (адрес) передачи заполненных бюллетеней инициатору;
- паспортные данные / номера квартир членов инициативной группы и кандидатов в совет дома.
